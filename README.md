# Sales and Revenue Analysis System using MySQL

## Project Overview

The **Sales and Revenue Analysis System** is a SQL-based database project designed to store, manage, and analyze business sales data.  
It helps in tracking customers, products, orders, and payments, and generates useful insights such as total revenue, monthly sales trends, and customer purchase behavior.

---

## Objectives

- Design a relational database for sales management  
- Track customer purchases and revenue  
- Analyze sales performance  
- Generate business insights using SQL queries  

---

## Technologies Used

- MySQL
- SQL (DDL, DML, Queries)
- MySQL Workbench

---

## Database Tables

### Customers
Stores customer information such as name, email, city, and registration date.

### Products
Contains product details including name, category, and price.

### Orders
Stores order transactions linked with customers.

### Payments
Tracks payment details and revenue generation.

---

## Key Features

- Tracks sales transactions  
- Calculates total revenue  
- Generates daily and monthly sales reports  
- Identifies high-value customers  
- Analyzes customer purchase patterns  

---

## SQL Concepts Used

- Primary Key and Foreign Key  
- Joins  
- Aggregation Functions (SUM, COUNT)  
- GROUP BY and ORDER BY  
- Date Functions
